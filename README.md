# AGRINHO2026* **
Estrutura Organizada (`index.html`):** O arquivo principal monta o esqueleto do site, contendo o título principal, os parágrafos explicativos e os blocos onde a imagem e o botão de interação ficam localizados.
* **2. Identidade Visual Verde (`style.css`):** O estilo define a aparência do site com tons de verde e cinza-escuro, remetendo à natureza e à limpeza, além de garantir que o layout se ajuste bem tanto em computadores quanto em celulares.
* **3. Imagem Temática:** Foi incluído um link direto para a foto de uma planta verde e saudável, usada para ilustrar visualmente como a flora depende diretamente de uma água limpa e livre de toxinas para sobreviver.
* **4. Interatividade (`script.js`):** O código em JavaScript adiciona uma função ao botão da página. Quando o usuário clica nele, o sistema processa a ação e exibe uma mensagem informativa na tela.
* **5. Foco Educativo:** O objetivo central do conjunto de arquivos é conscientizar sobre os impactos dos produtos químicos nos recursos hídricos e mostrar que a interrupção da poluição permite a recuperação dos ecossistemas.
